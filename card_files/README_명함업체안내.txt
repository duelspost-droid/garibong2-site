[ 규격 ]
- 표준 명함 90 x 50 mm
- 양면 (앞면 7종 + 공통 뒷면 1종)

[ 폴더 구성 ]

1) SVG_edit  ← 명함업체 편집용 (벡터 파일)
   - 일러스트레이터(Adobe Illustrator), 잉크스케이프(Inkscape)에서
     열어서 자유롭게 수정 가능합니다.
   - 텍스트, 색상, 위치 모두 편집됩니다.
   - 파일명:
       1_감사_신수호_앞면.svg
       2_이사_김명식_앞면.svg
       3_이사_서훈_앞면.svg
       4_이사_설희경_앞면.svg
       5_이사_김충식_앞면.svg
       6_이사_박은경_앞면.svg
       7_이사_한영택_앞면.svg
       공통_뒷면.svg  (전원 동일)

2) PDF  ← 인쇄 의뢰용 (90 x 50mm 정확한 규격)
   - 명함업체에 바로 전달 가능한 인쇄용 PDF입니다.
   - 페이지 크기가 정확히 명함 규격으로 맞춰져 있습니다.

3) JPG  ← 미리보기 / 시안 확인용 (300dpi)
   - 디자인 확인, 공유용 이미지입니다.

[ 폰트 안내 ]
- 제목/심볼 글씨: "Jua (주아)"  ※ 구글폰트 무료
   다운로드: https://fonts.google.com/specimen/Jua
- 본문 글씨: "Noto Sans KR"  ※ 구글폰트 무료
   다운로드: https://fonts.google.com/noto/specimen/Noto+Sans+KR
- SVG 편집 시 위 폰트를 PC에 설치하면 동일하게 보입니다.
- 인쇄 전 폰트를 "윤곽선/아웃라인(Create Outlines)"으로
  변환하시면 폰트 깨짐 없이 출력됩니다.

[ 색상 (CMYK 참고) ]
- 네이비:  #0d2461  (C100 M90 Y30 K30 근사)
- 블루:    #2563eb / #1a2a6c
- 회색:    #4b5563

[ 인쇄 권장 ]
- 재단 여백(도련) 1~2mm 추가 권장
- 해상도 300dpi 이상
