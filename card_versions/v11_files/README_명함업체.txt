============================================
 가리봉2구역재개발정비사업조합 명함 (v11)
============================================

[ 규격 ]
- 표준 명함 90 x 50 mm, 양면
- 앞면 7종(임원별) + 공통 뒷면 1종

[ 파일 ]
- front_1 ~ front_7 : 임원별 앞면
- back_common       : 공통 뒷면(풀 시그니처)
- 각 종마다 4가지 형식 제공:
  · SVG : 벡터 원본 (일러스트레이터/잉크스케이프 등에서 자유 편집·무한 확대)
  · PNG : 고해상 비트맵 (포토샵 작업용, 1800x1000px)
  · JPG : 고해상 이미지 (약 400dpi)
  · PDF : 90x50mm 인쇄용

[ 전체 묶음 파일 (8종을 한 파일로) ]
- all_cards_8page.pdf : 8페이지 PDF, 각 페이지가 명함 1장(90x50mm).
                        ★ 이 파일 하나만 전달하면 8종 전체가 인쇄 준비됩니다.
- all_cards.svg       : 8종이 한 벡터 파일(2열 x 4행). 일러스트레이터 편집용.
- all_cards_sheet.png / .jpg : 8종이 한 장에 보이는 확인용 시트(2열 x 4행).

[ SVG(벡터) 사용 안내 ]
- 글자가 폰트로 들어가 있어, 동일하게 보이려면 아래 서체를 먼저 설치하세요.
- 또는 업체에서 SVG의 텍스트를 윤곽선(outline/곡선)으로 변환하면
  폰트 설치 없이도 글자가 깨지지 않습니다. (인쇄 입고 시 권장)

[ 디자인 ]
- 앞면: 조합명(상단) + 서남권 최고 랜드마크 일러스트(좌) + 직책·이름·연락처(우)
- 뒷면: 심볼 + 조합명 + 슬로건 + 주소

[ 색상 (참고) ]
- 네이비: #16235c / #1a2a6c
- 골드(태양): #f59e0b
- 본문: #111 / #222

[ 서체 (무료, 구글폰트) ]
- 제목/조합명: Jua  → https://fonts.google.com/specimen/Jua
- 본문: Noto Sans KR → https://fonts.google.com/noto/specimen/Noto+Sans+KR

[ 인쇄 권장 ]
- 재단 여백(도련) 1~2mm 추가
- PDF는 90x50mm 단일 규격으로 제작되어 있습니다
- 색상은 CMYK 변환 후 교정 권장
